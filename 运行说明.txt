运行环境：
Python3
需要模块, 可能需要其它模块，根据提示安装
Django2.1.4
mysqlclient
PIL

数据库（在donghubbs\settings.py中设置）：
默认账号 root
默认密码 root
数据库名称 donghubbs

运行：(python也可能要改为python3)
1、cmd进入django项目路径下

2、python manage.py migrate #创建表结构，非model类的其他表，django所需要的

3、python manage.py makemigrations donghu #做数据迁移的准备
如：python manage.py makemigrations donghu
4. python manage.py createsuperuser # 根据提示创建管理员账号
5. python init_database.py # 初始化模块数据库，也可创建超级用户后在后台添加
6. python add_permission.py # 为每个模块添加相应的权限和组
7. python manage.py runserver # 运行

本地测试后台地址：
127.0.0.1:8000/admin
